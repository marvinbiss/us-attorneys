Classification des sources

Tier 1 — Bulk officiel, immédiatement actionnable

Datasets complets fournis directement par les autorités. Qualité maximale, bar numbers vérifiés.

États : NY (done), FL (email gratuit), GA ($6.5K)

Tier 2 — Demandes formelles (PIA/FOIA), délai 2-6 semaines

Demandes de records publics aux barreaux d'état. Données officielles, coût modéré.

États : TX ($200-500), OH (gratuit/faible), NJ ($200-500), PA (sur devis), MA (frais copie)

Tier 3 — Scraping testé et prouvé (fallback)

Méthodes testées avec 50-110 records. Fiable mais plus fragile à grande échelle.

États : CA (HTML scraping, ~14h), IL (ASP.NET grid, ~6-8h), TX (ColdFusion, ~8h)

Détails par état

Texas — Public Information Act (PIA)

Le State Bar of Texas est soumis au Texas PIA. Soumettre via le portail texasbar.govqa.us. Demander le bar roll complet en format électronique (CSV/Excel). Champs publics : nom, bar card number, adresse cabinet, téléphone, date de licence, statut. Les adresses personnelles et SSN sont confidentiels.

Alternative : Programme de mailing lists via listmail@texasbar.com (payant, prix non publié).

Alternative rapide : OpenGovUS (opengovus.com/texas-lawyer) semble avoir ~130K avocats scrapables.

Ohio — Public Records Request

Contacter le Registration and CLE Section de la Supreme Court of Ohio. L'Ohio Public Records Act (R.C. 149.43) oblige la fourniture des records électroniques dans leur format natif. Champs : nom, adresse pro, téléphone, n° enregistrement, statut, genre, date naissance, école de droit.

Note technique : Le domaine supremecourt.ohio.gov est inaccessible depuis notre machine (timeout). La demande officielle contourne ce problème.

New Jersey — CAMS Report

Le Central Attorney Management System (CAMS) des NJ Courts fournit des rapports de données d'avocats sur demande. Record layout PDF disponible. Mise à jour 2x/an. Restriction : interdiction de miner/revendre sans autorisation écrite.

Attention : Le portail de recherche en ligne interdit explicitement l'accès automatisé («May not be accessed by any automated program»).

Pennsylvania — Public Data Request

Le PA Disciplinary Board accepte les demandes de données publiques via un formulaire dédié, avec tarification au cas par cas. Champs : nom, statut (active/inactive/suspended/disbarred), adresse, historique disciplinaire, assurance malpractice.

Massachusetts — Chapter 66 Public Records Request

Le Board of Bar Overseers (BBO) est probablement soumis aux lois de records publics du Massachusetts (Chapter 66). Soumettre une demande formelle. Champs disponibles via leur lookup : nom, BBO#, adresse, date admission, historique disciplinaire.

C. Considérations légales

  • Public Records : Les bar rolls sont des documents publics dans la plupart des états. Exception : l'ARDC (IL) est exempt de FOIA en tant qu'organe de la branche judiciaire.
  • Scraping : Certains états interdisent explicitement l'accès automatisé (NJ : «automated program», FLSD : «commercial use prohibited»). Privilégier les demandes officielles.
  • Usage commercial : La FL Bar interdit les «unauthorized uses». La GA Bar exige un «one-time use». Vérifier les conditions pour chaque source.
  • Règle d'or : Mieux aucune information qu'une information fausse. Toute donnée d'avocat doit être traçable à sa source officielle.

D. Méthodologie

Cette analyse a été réalisée le 16 mars 2026 par 10 agents de recherche parallèles, couvrant :

  • Les 5 state bars prioritaires (CA, FL, TX, IL, OH)
  • 4 états supplémentaires (NJ, PA, MA, GA)
  • Les sources fédérales/nationales (PACER, CourtListener, USPTO, EOIR)
  • Les portails open data (data.gov, Kaggle, GitHub, Harvard Dataverse, Socrata)
  • Les fournisseurs commerciaux (Martindale, Avvo, ZoomInfo, Data Axle, etc.)

Chaque source a été évaluée sur : complétude, coût, format, fraîcheur, présence de bar numbers, et conditions d'utilisation.
