# 8. CADRE LEGAL : GESTION DE NOMS DE DOMAINE EN FRANCE

## 8.1 Reglementation AFNIC (.fr)

| Obligation | Detail |
|---|---|
| Registrar agree | Seul un registrar agree AFNIC peut enregistrer un .fr |
| Eligibilite titulaire | Siege social dans un Etat membre UE |
| Contact administratif | Obligatoire pendant toute la vie du domaine |
| Peut etre un tiers | Oui, le prestataire peut etre contact admin |
| Mise a jour coordonnees | Obligation de mise a jour sans delai |
| Protection donnees | Publication restreinte par defaut (RGPD/Whois) |
| Transfert | Le titulaire a le droit de transferer son domaine |

## 8.2 Points d'attention pour un service de gestion de sites clients

1. **Le client DOIT etre titulaire du domaine** : le prestataire ne doit pas enregistrer le domaine a son propre nom
2. **Droit de transfert** : le client doit toujours pouvoir recuperer son domaine
3. **Transparence** : les coordonnees du titulaire doivent etre exactes
4. **RGPD** : publication restreinte par defaut dans le Whois
5. **Pas de lock-in abusif** : pratique de Solocal (site desactive a fin de contrat) = tres critiquee

## 8.3 Bonnes pratiques
- Toujours enregistrer le domaine au nom du client
- Fournir les acces DNS au client
- Prevoir une clause de portabilite dans les CGV
- Utiliser un registrar francais agree AFNIC (OVH, Gandi, etc.)

# 10. SYNTHESE ET OPPORTUNITES

## Le marche en un coup d'oeil

| Metrique | Valeur |
|---|---|
| Artisans du batiment en France | 560 000 (CAPEB) |
| Sans site internet (~47%) | ~263 000 |
| Budget digital < 1 000 EUR/an | 74% des artisans |
| Concurrent principal (Simplebo) | 10 000 clients / 9M EUR CA |
| Concurrent #2 (Solocal) | 195 000 clients total / en declin |
| Prix acceptable creation | 400-800 EUR HT |
| Abonnement mensuel acceptable | 30-60 EUR HT |
| Churn estime | 4-6%/mois |
| CAC estime | 150-300 EUR |
| LTV estimee | 1 500-1 800 EUR |
| Ratio LTV:CAC | 5:1 a 12:1 |

## Opportunites identifiees

1. **Marche sous-penetre** : 263 000 artisans du batiment sans site = opportunite massive
2. **Concurrent faible en prix** : Simplebo a 45-60 EUR/mois minimum -> place pour une offre a 20-35 EUR/mois
3. **Solocal en declin** : perte de clients, reputation mediocre, lock-in = clients a recuperer
4. **Partenariats syndicaux** : CAPEB, FFB, CNAMS = canaux d'acquisition a faible cout
5. **Subventions** : artisans peuvent financer via aides regionales (30-80% du cout)
6. **Specificite metier** : un site "clef en main BTP" avec templates metiers = differenciation
7. **Facturation electronique 2027** : levier pour pousser la digitalisation

## Risques

1. **Conjoncture BTP defavorable** : -5% d'activite, budgets contraints
2. **Churn eleve SMB** : 4-6%/mois = renouvellement constant necessaire
3. **Simplebo bien installe** : 8 ans de partenariat CAPEB, forte satisfaction
4. **Budget digital faible** : 43% des artisans depensent < 300 EUR/an
5. **IA disruptive** : outils IA gratuits/low-cost pourraient commoditiser la creation de sites
