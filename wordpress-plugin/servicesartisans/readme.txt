Affichez votre badge "Artisan Verifie" ServicesArtisans sur votre site WordPress. Note, avis et statut en temps reel.

== Description ==

**ServicesArtisans** est le premier annuaire d'artisans verifies en France. Ce plugin gratuit vous permet d'afficher votre badge de verification directement sur votre site WordPress.

= Fonctionnalites =

* **Badge dynamique** — Affiche votre note, nombre d'avis et statut de verification en temps reel
* **3 styles** — Clair, Sombre ou Minimal pour s'adapter a votre theme
* **Bouton devis** — Ajoutez un bouton "Demander un devis gratuit" pour convertir vos visiteurs
* **Widget sidebar** — Glissez-deposez le widget dans votre barre laterale
* **Shortcodes** — Integrez le badge n'importe ou avec `[servicesartisans_badge]`
* **Ultra-leger** — Aucun JavaScript, juste une image SVG de moins de 3 Ko
* **Compatible** — Fonctionne avec tous les themes et constructeurs de pages (Elementor, Divi, Gutenberg)

= Pourquoi afficher le badge ? =

* **Confiance client** — 73% des internautes font plus confiance aux sites avec un badge de verification
* **Plus de demandes** — Les artisans avec badge recoivent en moyenne +35% de contacts
* **SEO** — Le badge ameliore votre referencement naturel
* **Mise a jour auto** — Votre note et avis se mettent a jour automatiquement

= Comment ca marche ? =

1. Installez et activez le plugin
2. Allez dans **Reglages > ServicesArtisans**
3. Entrez votre identifiant (slug de votre fiche)
4. Le badge apparait automatiquement via le widget ou les shortcodes

= Shortcodes disponibles =

* `[servicesartisans_badge]` — Affiche le badge avec les reglages par defaut
* `[servicesartisans_badge style="dark"]` — Badge en mode sombre
* `[servicesartisans_badge devis="non"]` — Badge sans bouton devis
* `[servicesartisans_devis]` — Bouton devis uniquement
* `[servicesartisans_devis texte="Devis gratuit" style="outline"]` — Bouton personnalise

== Installation ==

1. Telechargez le fichier ZIP du plugin
2. Dans votre WordPress, allez dans **Extensions > Ajouter > Televerser**
3. Selectionnez le fichier ZIP et cliquez sur **Installer maintenant**
4. Activez le plugin
5. Configurez votre identifiant dans **Reglages > ServicesArtisans**

== Frequently Asked Questions ==

= Le plugin est-il gratuit ? =

Oui, 100% gratuit, sans frais caches ni abonnement.

= Le plugin ralentit-il mon site ? =

Non. Le badge est une simple image SVG de moins de 3 Ko chargee depuis nos serveurs (CDN). Aucun JavaScript n'est ajoute a votre site.

= Comment trouver mon identifiant ? =

Votre identifiant (slug) se trouve dans l'URL de votre fiche sur ServicesArtisans.fr. Par exemple, si votre fiche est a l'adresse `servicesartisans.fr/services/plombier/paris/dupont-plomberie`, votre identifiant est `dupont-plomberie`.

= Le plugin fonctionne-t-il avec Elementor / Divi / Gutenberg ? =

Oui, le plugin est compatible avec tous les constructeurs de pages. Utilisez le shortcode `[servicesartisans_badge]` dans un bloc HTML ou texte.

= Puis-je personnaliser le badge ? =

Vous pouvez choisir entre 3 styles (Clair, Sombre, Minimal) et activer/desactiver le bouton devis. Les donnees (note, avis, verification) sont tirees automatiquement de votre fiche.
